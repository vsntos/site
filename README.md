# vsantos.rbind.io

Site pessoal e portfólio acadêmico de Vinicius Santos.

## Stack

- [Hugo](https://gohugo.io/) com o tema [Wowchemy Academic](https://wowchemy.com/)
- Hospedado no [GitHub Pages](https://pages.github.com/)
- Deploy automático via GitHub Actions

## Desenvolvimento local

```bash
hugo server
```

O site estará disponível em `http://localhost:1313`.

## Estrutura

```
content/
  authors/admin/   # Perfil do autor
  home/            # Seções da página inicial
  post/            # Posts e análises
  publication/     # Publicações acadêmicas
  talk/            # Apresentações
  courses/         # Materiais de cursos
```
